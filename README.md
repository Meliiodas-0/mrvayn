# MrVayn | Game Developer Portfolio

Showcasing my journey, experience, and projects.

## Project Structure

- `src/components`: UI components and sections.
- `src/pages`: Main application pages.
- `src/assets`: Project assets.
- `public`: Static files.

## Tech Stack

- **React**: Frontend library.
- **Vite**: Build tool.
- **TailwindCSS**: Styling.
- **Framer Motion**: Animations.
- **Three.js**: 3D effects.

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
